🔥 Firebase Security Rules 설정 가이드

장비 관리 페이지 권한 오류 해결 방법

현재 문제

MANAGER 역할로 로그인 후 "장비 관리" 페이지에서 장비 목록을 불러올 수 없습니다.
콘솔 에러: FirebaseError: Missing or insufficient permissions.

해결 방법

1단계: Firebase Console 접속

🔗 https://console.firebase.google.com/ 접속
프로젝트 선택: hvac-management-477fb

2단계: Firestore Database 메뉴 이동

왼쪽 메뉴에서 "Firestore Database" 클릭
상단 탭에서 "Rules" (규칙) 클릭

3단계: 기존 규칙 확인

현재 equipment 컬렉션 규칙이 다음과 같이 설정되어 있는지 확인:

match /equipment/{equipmentId} { allow read: if isAuthenticated(); allow create, update: if isManagerOrAbove(); allow delete: if isAdmin(); }

4단계: 전체 규칙 복사 및 적용

아래 규칙을 복사하여 Firebase Console의 Rules 탭에 붙여넣고 "게시" 버튼을 클릭하세요:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // 헬퍼 함수: 사용자가 인증되었는지 확인 function isAuthenticated() { return request.auth != null; } // 헬퍼 함수: 사용자가 관리자인지 확인 function isAdmin() { return isAuthenticated() && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin'; } // 헬퍼 함수: 사용자가 점검자 이상인지 확인 function isInspectorOrAbove() { return isAuthenticated() && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role in ['inspector', 'manager', 'admin']; } // 헬퍼 함수: 사용자가 관리자 이상인지 확인 function isManagerOrAbove() { return isAuthenticated() && get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role in ['manager', 'admin']; } // 사이트(현장) 컬렉션 match /sites/{siteId} { allow read: if isAuthenticated(); allow create, update: if isManagerOrAbove(); allow delete: if isAdmin(); } // 건물 컬렉션 match /buildings/{buildingId} { allow read: if isAuthenticated(); allow create, update: if isManagerOrAbove(); allow delete: if isAdmin(); } // 장비 컬렉션 ⭐ 중요 match /equipment/{equipmentId} { allow read: if isAuthenticated(); allow create, update: if isManagerOrAbove(); allow delete: if isAdmin(); } // 점검 기록 컬렉션 match /inspections/{inspectionId} { allow read: if isAuthenticated(); allow create: if isInspectorOrAbove(); allow update: if isInspectorOrAbove() && (request.auth.uid == resource.data.inspector_id || isManagerOrAbove()); allow delete: if isAdmin(); } // 점검자 컬렉션 match /inspectors/{inspectorId} { allow read: if isAuthenticated(); allow create, update, delete: if isManagerOrAbove(); } // 사용자 컬렉션 ⭐ 회원가입 허용 match /users/{userId} { allow read: if isAuthenticated(); allow create: if request.auth != null && request.auth.uid == userId; allow update: if isAuthenticated() && (request.auth.uid == userId || isAdmin()); allow delete: if isAdmin(); } } }

5단계: 규칙 게시

Firebase Console에서 "게시" (Publish) 버튼을 클릭하여 규칙을 적용합니다.
⚠️ 게시하지 않으면 변경사항이 적용되지 않습니다!

설정 완료 후 확인

1. 브라우저에서 페이지 새로고침 (Ctrl + F5 또는 Cmd + Shift + R)
2. MANAGER 계정으로 로그인
3. "장비 관리" 페이지 접속
4. 장비 목록이 정상적으로 표시되는지 확인

보안 정책 요약

컬렉션 읽기 생성/수정 삭제
users 모든 인증 사용자 본인 또는 ADMIN ADMIN
equipment 모든 인증 사용자 MANAGER, ADMIN ADMIN
inspections 모든 인증 사용자 INSPECTOR, MANAGER, ADMIN ADMIN
sites 모든 인증 사용자 MANAGER, ADMIN ADMIN
