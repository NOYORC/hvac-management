데이터 이름 필드 업데이트

기존 데이터에 가독성을 위한 이름 필드를 추가합니다

업데이트 내용:
  • Buildings: site_id → site_name 추가
  • Equipment: site_id, building_id → site_name, building_name 추가

목적: ID만으로는 알기 어려운 정보를 이름으로 즉시 확인 가능하게 합니다.

주의사항:
  • 이 작업은 기존 데이터를 수정합니다
  • 이미 이름 필드가 있는 경우 최신 정보로 업데이트됩니다
  • 작업 중에는 페이지를 새로고침하지 마세요
  • 완료까지 1-3분 정도 소요될 수 있습니다
